music-player (app)

The HTML Audio API provides an interface to manipulate audio files with the help of a few JavaScript libraries. For this little app, I was experimenting on a music player and I was able to come up with something that extracts MP3 metadata from ID3 metadata container, a de facto standard for MP3 files. This App can be used as building blocks to creating Apps like Soundcloud and Bandcamp

img-processor

Sometimes I just like to play around a few html native APIs. This little app does basic image processing and can be used as builidng blocks to create advanced image processing tools similar to Instagram Editor and other image editing tools

drag-and-drop

This sortable list app utilises the html drag and drop API to sort through a list of my favourite songs. It's a short script that allows you to sort through the list while the checking if the order is correct. This API can be used as building blocks to create complex projects such as task management and collaboration applications as well as interactive image galleries.
